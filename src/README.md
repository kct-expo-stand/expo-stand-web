# Папка SRC

Эта папка предназначена для размещения исходных файлов проекта, которые могут требовать компиляции или транспиляции.

## Возможное использование:

- TypeScript файлы (если проект будет расширен до TypeScript)
- SCSS/SASS файлы для препроцессинга CSS
- Исходники для минификации и оптимизации
- Модули для сборки с помощью Webpack/Vite/Rollup

## Текущий статус:

На данный момент проект использует простые HTML/CSS/JS файлы без сборки.
Эта папка зарезервирована для будущего расширения функциональности.
